In this case dhtmlxWindows will create a new object and use it as a viewport.

Besides, this viewport can be attached to any existing object on page. It has absolute position.

By default the viewport has no border. In our samples we showed the border to specify the boundaries
in which the window is displayed on page, i.e. we visualised the viewport.
